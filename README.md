# Mini Project Report: Accelerating Materials Discovery using Deep Learning

This repository hosts the web-based project report for my B.Tech Mini Project (CO4015). 

The project evaluates modern machine learning frameworks for predicting material properties and atomic forces in Molecular Dynamics (MD) simulations. It is divided into two primary phases:

1. **Phase 1: CGCNN** - An exploration into predicting macroscopic material properties directly from crystal structure graphs.
2. **Phase 2: ACE vs MACE** - A rigorous benchmark comparing localized Atomic Cluster Expansion against equivariant message-passing networks (MACE) for force predictions.
